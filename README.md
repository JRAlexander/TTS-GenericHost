# TTS-GenericHost

IMPORTANT:

 * Requires the following Nuget Packages: 
 *      Microsoft.Extensions.Hosting, Version 2.1.1
 *      Microsoft.Extensions.Hosting.Abstractions, Version 2.1.1
 *      Microsoft.Extensions.Http, Version 2.1.1
